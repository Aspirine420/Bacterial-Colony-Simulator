# 🦠 Bacterial Colony Simulator

Минималистичный симулятор эволюции и выживания бактериальных колоний на Python (Pygame). Наблюдайте за сложными взаимодействиями пяти различных видов в замкнутой экосистеме.

## 🚀 Особенности видов

*   **🟩 Зеленые (Рост):** Быстро захватывают территорию. Находятся в симбиозе с Грибницей, но получают ожоги от Желтых санитаров.
*   **🟦 Голубые (Энергетики):** Самый чистый вид. Имеют иммунитет к яду, живут долго и питаются старыми Желтыми клетками.
*   **🟥 Красные (Хищники):** Охотятся на Зеленых, но сильно загрязняют среду отходами. Опасаются яда Желтых.
*   **🟨 Желтые (Санитары):** Поглощают фиолетовый токсичный яд, очищая поле. Смертельно опасны для Зеленых при контакте.
*   **🟫 Грибница:** Защитный барьер. Убивает всех, кроме Зеленых. Редко выпускает споры, которые заражают новые участки поля.

## 🎮 Управление

*   **Клавиши [1] - [5]:** Выбор типа бактерии для размещения.
*   **ЛКМ (зажать):** Сажать выбранный тип (режим кисти).
*   **ПКМ (зажать):** Стирать бактерии (ластик).
*   **Space:** Пауза / Запуск симуляции.
*   **C:** Полная очистка поля.
